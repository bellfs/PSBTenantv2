## Redaction Standard

- Personal tenant/prospect names, phone numbers, private emails, one-time codes and IDs are not intentionally stored in this curated canon.
- Business emails, company names, property addresses, public supplier names and strategic operating facts may be stored where operationally useful.
- If a sensitive fact is needed for action, the agent should retrieve it from the live source system or uploaded document rather than from this file.
